Save changes using Git and Github

Now that you have finished your program, it is time to save it in Github. We do this so that potential employers can see your work. It is good to be in the habit of saving your code in Github even in the very beginning. Anytime you write a program or work on a project, you will want to do what we are about to do now. There are two simple ways to do this. One is using the VS Code Graphical User Interface, the other is using the terminal.

  • Click on the Source Control icon on the side of VS Code (Under the file explorer and search icons)

  • Enter a message (shown with red rectangle in image above). This will be your commit message which is meant to be helpful for you to remember what you did in your code (For example, "Completed Assignment 1", or "Completed first task, need to add CSS"). Notice the controls to save this, most like either ctrl+enter or cmd+enter.

  • Select ALWAYS when prompted if you want to always add your files to your commits automatically.

  • Click on the three dots, then click push.

  • Now, you should be able to go back to github.com, refresh the page, and see your changes there.

  • If VS Code is unable to complete this process, please open a terminal window (by clicking terminal at the top of the page and selecting new terminal). In there, you will need to execute the following commands (with your email and name used when you created your github account) then restart visual studio code.

    • git config --global user.name "John Doe"

    • git config --global user.email "johndoe@example.com"

  • Open the terminal in VS code by right clicking the JavaScript1 folder and selecting "Open in Integrated Terminal".

  • execute the following commands:

  • git status - This will show what changes have been made in your repository.

  • git add . - This will prepare all of your changes for a commit (which will be saved in version control if you ever need to view or return to the state that your code is currently in).

  • git commit -m "Lesson # assignment" - This will save the changes that you just added with the name that you put for the message, in this case: "Lesson 1 Assignment".

  • git push - This will save your code in Github. Now you could go to the github.com website and view your code their as well along with your commit.

  • Navigate to github.com and verify that your repository shows your latest code.
